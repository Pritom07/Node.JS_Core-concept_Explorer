# Node.JS_FileSystem_Explorer

This repository contains two core Node.js practice modules:

1. **Different_Modules**
2. **File_Organizer_using_CLI**

Each folder is independent and focuses on a different concept of Node.js.  
It's a **high-level overview** — detailed documentation is inside each folder.
